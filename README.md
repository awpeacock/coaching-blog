# Coaching Blog

A very basic blog built in React (using Vite).

At present, content is hardcoded and stored locally. Longer term, it is to be hosted on AWS (S3, Route53 and CloudFront via CloudFormation), using a Contentful backend for retrieving the content.

## Status

This project is in an early, static prototype phase.

## Installation

To install all dependencies, simply run:

```
npm install
```

## Running locally

To build and run the site locally, execute the following command:

```
npm run dev
```
